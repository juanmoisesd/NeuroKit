# Ethics Statement

## Research Ethics

This software distribution adheres to the following ethical principles:

### Integrity & Attribution
- All software logic and methodologies from the original NeuroKit2 project are properly attributed.
- Modifications made for technical stabilization and reproducibility are documented.

### Privacy
- This repository is a software distribution and does not contain primary primary data involving human subjects.
- Included example data is derived from the original NeuroKit2 project's anonymized samples.

### Transparency
- The provenance of this distribution is tracked via `related_identifiers.json` and formal metadata records.

### Responsible Use
- This software is released under the MIT License to maximize scientific utility.
- Users are encouraged to cite both this distribution and the primary NeuroKit2 publication.
