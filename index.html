📦 NeuroKit2

The Python Toolbox for Neurophysiological Signal Processing

📖 About This Software

NeuroKit2 is an open-source Python toolbox designed for the comprehensive processing and analysis of neurophysiological signals (ECG, EEG, EDA, EMG, PPG). This distribution is part of a reproducibility initiative aimed at standardizing physiological data analysis.
