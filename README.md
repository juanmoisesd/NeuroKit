## Overview
This repository contains the curated distribution of **NeuroKit2**, focusing on reproducibility and standardization in neurophysiology, neuroeconomics, and fraud detection.

## Structure
- `neurokit2/`: Core library source code.
- `docs/`: Documentation and tutorials.
- `tests/`: Comprehensive test suite.
